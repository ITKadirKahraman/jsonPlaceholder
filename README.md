# 🌐 API Playground

Ein persönliches Lern- und Testprojekt, das verschiedene **REST-APIs** in einer einzigen Webanwendung zusammenführt. Ziel dieses Projekts ist es, den Umgang mit unterschiedlichen APIs zu erlernen, Daten dynamisch abzurufen und moderne JavaScript-Techniken wie `fetch()`, `async/await` und JSON-Verarbeitung praxisnah einzusetzen.

Dieses Repository dient als zentrale Sammlung für API-Experimente und wird kontinuierlich um neue Schnittstellen und Funktionen erweitert.

---

# 🚀 Projektziel

Das Ziel dieses Projekts ist es, möglichst viele öffentliche APIs in einer Anwendung zu integrieren und dabei unterschiedliche Anwendungsfälle kennenzulernen.

Dabei stehen folgende Themen im Fokus:

- Arbeiten mit REST-APIs
- HTTP-Methoden (GET, POST, PUT, DELETE)
- JSON verarbeiten
- Asynchrone Programmierung
- Dynamisches Rendering von Daten
- API-Authentifizierung
- Fehlerbehandlung
- Pagination
- Suchfunktionen
- Responsive Webentwicklung

# 📚 Warum dieses Projekt?

Mit diesem Repository möchte ich meine Kenntnisse im Bereich der Frontend-Entwicklung kontinuierlich erweitern und praktische Erfahrung im Umgang mit Web-APIs sammeln.

Das Projekt dient gleichzeitig als Lernplattform und Portfolio-Projekt, an dem neue Technologien und API-Konzepte ausprobiert und dokumentiert werden.
